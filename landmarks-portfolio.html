Landmarks

Last summer, I wanted to try my hand at mobile development. I decided to create an app that would help me learn about the landmarks across the Country. I used the Google Maps API to create a map that would show the user the location of the landmark. I also used the Wikipedia API to get information about the landmark. I used SwiftUI to create the app and I used the MVVM design pattern to organize my code. I also used the Combine framework to handle asynchronous calls to the APIs. I learned a lot about mobile development and I am excited to continue learning more about it.
